# Liquid-Engine-Project
Liquid Engine final year project

######This repository is created to organize our final year project of Bipropellant Liquid Engine Design#####
This project contains following phases
1. Design study and numerical analysis using RPA and NASA CEA software
2. Custom python script developement for iterations and optimization and design calculation
3. CAD Mmodelling of various subsystems
4. BOM and manufacturing study of all subsystems
5. Heat transfer, nozzle cfd, injector simulations and validation
6. Manufacturing and testing
